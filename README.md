# CollabFind

**CollabFind** adalah platform kolaborasi inovatif yang dirancang untuk menghubungkan para profesional, developer, desainer, dan kreator dengan proyek-proyek menarik dan kolaborator yang tepat. Temukan tim impian Anda, kelola proyek dengan efisien, dan bangun portofolio yang mengesankan di satu tempat.

## ✨ Fitur Utama

- **Pencocokan Cerdas (Smart Match):** Algoritma canggih kami membantu Anda menemukan proyek atau talenta yang paling sesuai dengan keahlian dan minat Anda.
- **Ruang Kerja Kolaboratif (Workspace):** Setiap proyek dilengkapi dengan ruang kerja terintegrasi yang mencakup:
  - **Papan Proyek (Kanban):** Visualisasikan alur kerja dan lacak kemajuan tugas dengan papan Kanban yang intuitif.
  - **Obrolan Tim (Team Chat):** Berkomunikasi secara real-time dengan anggota tim Anda.
  - **Penyimpanan File (File Storage):** Bagikan dan kelola aset proyek dengan mudah.
  - **Wiki Proyek:** Dokumentasikan ide dan pengetahuan penting dalam satu basis data terpusat.
- **Editor Portofolio Dinamis:** Bangun dan pamerkan portofolio profesional Anda untuk menarik perhatian perekrut dan kolaborator.
- **Eksplorasi Proyek:** Jelajahi beragam proyek dari berbagai kategori dan industri.
- **Komunitas Aktif:** Terhubung dengan sesama profesional melalui forum, acara, dan hackathon.
- **Manajemen Tugas:** Atur sprint, tetapkan tugas, dan pantau progres tim Anda.

## 🚀 Tumpukan Teknologi

CollabFind dibangun menggunakan tumpukan teknologi modern yang cepat, andal, dan skalabel:

- **Frontend:** [React](https://react.dev/) & [Vite](https://vitejs.dev/)
- **Backend & Database:** [Supabase](https://supabase.io/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Routing:** [React Router](https://reactrouter.com/)
- **Animasi:** [Framer Motion](https://www.framer.com/motion/)
- **Ikon:** [Lucide React](https://lucide.dev/)

## 🛠️ Instalasi & Menjalankan Proyek

Untuk menjalankan proyek ini secara lokal, ikuti langkah-langkah berikut:

1.  **Clone repositori ini:**
    ```bash
    git clone https://github.com/username/collabfind.git
    cd collabfind
    ```

2.  **Instal dependensi:**
    ```bash
    npm install
    ```

3.  **Konfigurasi Environment Variables:**
    Buat file `.env` di root proyek dan tambahkan kunci Supabase Anda.
    ```
    VITE_SUPABASE_URL=URL_SUPABASE_ANDA
    VITE_SUPABASE_ANON_KEY=KUNCI_ANON_SUPABASE_ANDA
    ```

4.  **Jalankan server pengembangan:**
    ```bash
    npm run dev
    ```

    Aplikasi akan berjalan di `http://localhost:5173` (atau port lain yang tersedia).
